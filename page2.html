Лабораторна робота №2

Тема:

КАСКАДНІ ТАБЛИЦІ СТИЛІВ. СЕЛЕКТОРИ .ІДЕНТИФІКАТОРИ. СТИЛЬОВЕ ОФОРМЛЕННЯ ТЕКСТОВИХ ЕЛЕМЕНТІВ В HTML-ДОКУМЕНТАХ.

Мета:

придбати практичні навички роботи з селекторами, ідентифікаторами, списками, різноманітними властивостями кольору і фону, зовнішними та внутрішними відступами, плаваючими елементами, оформленням текстових елементів

ХІД ВИКОНАННЯ РОБОТИ

1. Селектори тегу

<!-- HTML -->
<p>Це селектор тегу для <b>абзацу</b>.</p>

<!-- CSS -->
p {
    color: blue;
}

2. Селектори класів

<!-- HTML -->
<div class="highlight">Це селектор класу.</div>

<!-- CSS -->
.highlight {
    background-color: yellow;
    padding: 10px;
}

3. Селектори ідентифікаторів

<!-- HTML -->
<h1 id="main-title">Це селектор ідентифікатора</h1>

<!-- CSS -->
#main-title {
    font-size: 24px;
    color: green;
}

4. Інші селектори

<!-- HTML -->
<ul>
    <li>Елемент 1</li>
    <li class="special">Елемент 2</li>
    <li>Елемент 3</li>
</ul>

<!-- CSS -->
ul li + li {
    color: red;
}

ul li.special {
    background-color: lightblue;
}

ul li:last-child {
    font-weight: bold;
}

5. CSS: Шрифти Текст Таблиці Фон Контур Списки СSS Просунутий

<!-- HTML -->
<div class="styled-text">Це стильовий текст.</div>

<!-- CSS -->
.styled-text {
    font-family: "Arial", sans-serif;
    color: purple;
    background-color: #e0e0e0;
    padding: 10px;
    border: 1px solid black;
}
